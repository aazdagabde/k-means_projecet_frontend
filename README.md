## ðŸ“‹ Overview

The **SmartSeg AI Dashboard** is the client-side interface for the SmartSeg segmentation platform. Built with **React** and **TypeScript**, it provides a seamless experience for uploading customer data, visualizing K-Means clustering results, and exploring AI-driven marketing strategies generated by Google Gemini.

This project emphasizes **UX/UI best practices**, interactive data visualization, and reporting capabilities.

## âœ¨ Key Features

### 1. Data Ingestion
* **Drag & Drop Interface:** Intuitive CSV file upload with format validation.
* **Dynamic Configuration:** Real-time adjustment of `K` (number of clusters) and CSV separators.

### 2. Interactive Visualization
* **RFM Scatter Plot:** Visualizes *Monetary* vs. *Frequency* with toggleable Log/Linear scales using `Recharts`.
* **Cluster Distribution:** Interactive Pie Charts showing segment sizes.
* **KPI Indicators:** Visual cues for trend analysis (vs. global averages).

### 3. Reporting & Persistence
* **HD PDF Export:** Generates professional multi-page reports with charts and AI insights using `jspdf` and `html2canvas`.
* **Analysis History:** Browses past segmentations stored in MongoDB via the backend API.

## ðŸš€ Getting Started

### Prerequisites
* Node.js (v16 or higher)
* npm or yarn

### Installation

1.  **Navigate to the client directory:**
    ```bash
    cd client
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the Development Server:**
    ```bash
    npm run dev
    ```
    The application will run on `http://localhost:5173`.

To create an optimized production build:
